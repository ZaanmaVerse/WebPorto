Capture The Flag

This Capture The Flag (CTF) event is designed as a comprehensive learning platform to develop and assess practical skills across various domains of cybersecurity. The challenges are categorized into key areas, each representing a distinct field of expertise. The Web Exploitation category includes tasks such as gottagofast, broken-eh, and Recruitprogrammer, which focus on identifying and exploiting vulnerabilities in web applications, including SQL Injection, Command Injection, and misconfigurations in authentication and authorization mechanisms. The OSINT (Open Source Intelligence) category, with challenges like Find My Friend and namejumpheadbang, emphasizes the ability to gather information from publicly accessible sources such as social media and websites, closely reflecting real-world cyber investigation techniques. Meanwhile, the Cryptography category, represented by challenges like XORry and Triple Threat, engages participants in solving both classical and modern encryption puzzles, often involving weaknesses in symmetric ciphers and flawed RSA implementations. The Forensics category immerses participants in complex digital investigation scenarios involving disk images (AD1), network captures (PCAP), and multimedia files, where they apply skills such as metadata analysis (EXIF), data carving, steganographic detection, and log interpretation. The Miscellaneous category introduces logic-based and non-conventional challenges that test participants' lateral thinking and creativity beyond standard technical frameworks. Overall, this CTF is structured to build critical competencies such as vulnerability analysis, manual exploitation, and proficiency with industry-standard tools including Burp Suite, Wireshark, FTK Imager, Steghide, Python, and HxD. Participants not only learn how to detect and understand system weaknesses but also practice safe exploitation techniques to retrieve sensitive data such as flags. With its holistic and realistic approach, this CTF serves as an ideal medium for anyone seeking to deepen their hands-on understanding of cybersecurity from offensive, defensive, and investigative perspectives.

Phishing Detector

Phishing Detector is a Machine Learning-based project developed to detect phishing attacks through both URLs and email content. This project aims to help individuals and organizations identify potential cyber threats automatically and efficiently. The system is built using the Python programming language, utilizing Machine Learning algorithms (Scikit-learn) to learn patterns from existing phishing datasets. For URL detection, features such as URL length, the number of suspicious symbols, and domain structure are analyzed to classify whether a link is harmful or safe. For email detection, the system processes the text content of emails using Natural Language Processing (NLP) techniques and TF-IDF vectorization to identify phrases or sentences commonly used in phishing.

Cyber Risk Assessment

A web-based application designed to help organizations systematically identify and assess cybersecurity risks based on the NIST SP 800-30 Rev.1 framework. It features interactive risk input, dynamic risk visualization (Low–High), and PDF report export. Built using Python (Flask), SQLite, and Chart.js, this tool is ideal for IT security teams and compliance divisions.

Wumpus Game

Wumpus Game is a grid-based game that I developed based on the classic concept from the world of artificial intelligence, namely Wumpus World. In this game, players control an agent or character who explores a mysterious cave divided into squares. Within the cave, there are various elements such as gold that must be found, Wumpus as a deadly monster that must be avoided, and holes that can trap the player. The game is designed with the main challenge being to make players use logic and indirect clues, such as the foul smell near Wumpus or the wind near the holes, to guess the location of dangers and make safe decisions. Players win if they successfully find the gold and exit safely, while the game ends if the player is trapped in a hole or eaten by the Wumpus. This project combines elements of logic programming, grid systems, and simple UI/UX, making it an excellent exercise in implementing AI concepts in the form of an interactive game.
